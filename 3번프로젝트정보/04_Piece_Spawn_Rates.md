## 특이사항

- **Knight와 Bishop**: 동일한 비용(3)으로 항상 같은 확률
- **Rook과 King**: 동일한 비용(5)으로 항상 같은 확률  
- **Queen**: 가장 비싼 기물(비용 9)로 레벨 9부터 등장
- **레벨 2**: 해금되는 기물이 없어 Pawn만 100%

## 확률 계산 공식

```
최종 가중치 = (1/기물비용) × (1-레벨진행도) + 1.0 × 레벨진행도

레벨진행도 = (현재레벨 - 1) / (최대레벨 - 1)
          = (현재레벨 - 1) / 9

각 기물 확률 = (해당 기물 가중치 / 전체 가중치 합) × 100%
```

*주의: 최대 레벨(maxLevel)은 추정값입니다. 실제 게임에서는 다를 수 있습니다.*
