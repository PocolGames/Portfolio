# 디자인 기본 설명 문서

2d 디자인으로만 진행하면 됨.

모든 디자인 방향이 확실하게 정해진 것이 아니기 때문에 디자이너가 고민했을 때, 추천하고 싶은 방향이 있으면 언제든지 이야기를 해주면 좋을 것 같음.

지금 확실하게 정해진 것은 [그리드 타일맵](/docs/01_Design_Planning_Documents/01_desigin_map_reference_explain.md)이 있다.

## 그리드 타일맵

타일맵은 주기적으로 계속 새로 디자인하여 판매할 계획이 있다. 

주기적으로 DLC 또는 인게임 재화로 구매할 수 있는 맵을 제작할 계획이 있으니 담당 디자이너는 `디자이너를 위한 타일맵 가이드 라인 문서`를 제작하는 것을 권장한다.

## 기물 디자인

현재 인게임에서 테스트 용으로 사용중인 [기물 디자인](/docs/02_Work_Log_Documents/01_design_test_image.md)은 진짜 테스트 용 이미지다.

기물들의 방향과 팀 별 구분 방법은 아직 존재하지 않고 있다.

디자이너들은 비주얼적으로 괜찮은 방향이 있으면 이야기를 해주면 좋을 것 같음.

예를들어, 이동하면서 바라보는 방향을 바꾸는 편이 자연스러운지, 팀 별 구분 방법은 어떻게 할 것이지. 스타크래프트처럼 피아식별 띠 같은 것을 사용할지, 팀 별로 각각 따로 디자인을 할 것인지 고민해봐야됨.
