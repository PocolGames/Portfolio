## 회의 내용

### 전준호

저번에 타일 시스템, 타일 띄워주고 숫자 띄워주는 것 다 하고, 추가로 타일 시스템에서 필수적으로 필요하게 될 효과들을 만드는 중, 빙고 시스템과 적 기물을 잡았을 때, 타격감을 책임지게 될 이펙트를 만드는 중.

> 개발 순서를 조정할 필요가 있어 보인다. 첫 번째 목적은 빠르게 플레이가 가능하게 만드는 건데, 필수 요소가 있음. 이거 없으면 플레이가 안 된다. 이거 좀 신경 쓰면서 하면 좋을 것 같다.
> 타격감을 책임지게 될 이펙트. 이거 좀 많이 중요한데, 없다고 플레이가 안 되는 건 아님.
> 시야 시스템을 먼저 하는 방향으로 가야 될 것 같다.
> 배치 가능 영역



### 최광남

저번 주에 체스 말에 대한 기본적인 시스템 틀을 만들어놨는데, 그거로 이번에 배치해서 해당 타일을 
타일에 배치를 하고 배치를 기준으로 이동 가능한 범위를 보여주고 클릭하면 이동하는 것을 만들 예정입니다.

-> 할 일: 범위 보여주고 클릭하면 이동하는 것.

네트워크 이동을 할 때, 어떤 이동을? 버튼 눌렀을 때 해당 타일로 이동 기능.

> 해당 타일로 이동할 때, 이동 방식은 확실하게 정해지지 않아서 순간 이동이라든지, 보간 적용된 거 없앤다든지, 지금은 쉽게 적용된 것으로 보이는데, 이런 거는 프로토타입을 만드는 데 지장이 없기 때문에 나중으로 미뤄도 상관없을 것 같다.
