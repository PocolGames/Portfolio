# 안티타일링과 안개맵

## 주요 논의사항 및 결정사항

### 1. 타일 잡음 추가 (Anti-Tiling/Tiling Artifact Reduction)
현재 상황
- 단순 반복 패턴의 타일링 아티팩트가 눈에 띄는 문제 발생
- 디자이너가 여러 변형 타일을 제작해야 하는 상황

검토 필요사항
- 준호의 기술적 해결 가능 여부 확인 필요
- 해결 시 디자이너 작업량 대폭 감소 예상

### 2. 디테일 타일 작업
결정사항
- 반복 패턴이 아닌 특수 디테일 타일은 디자이너가 별도 작업 진행

### 3. 안개맵 디자인 개선
현재 문제점
- 현재 안개가 평면적이고 부자연스러움 (임시 버전)

향후 계획
- 프로그래머 측 연구 필요
- 두꺼운 안개맵 적용 시 타일 침범 문제 해결 방안 모색
- 디자이너가 비주얼적으로 만족스러운 안개 디자인 먼저 제작

### 4. 디자이너 남은 작업 일정
진행 방향
- 모든 기본 디자인 요소 완료 후 디테일 작업 진행
- 우선 작업: 안개 디자인, 스도쿠 번호 디자인, UI 디자인

### 5. 추가 검토사항
빙고 완성 시 타일 디자인 변경
- 구현 가능 여부 준호 검토 필요

## 프로그래머 담당 업무 (준호)

### 타일 잡음 추가
- 몇 가지 시도 가능한 방법 보유
- 결과 불확실성으로 인한 테스트 우선 진행

### 안개맵 개선
- 후순위 연구 과제
- 디자이너 디자인 시안이 방향성 결정에 도움될 예정

### 기술적 고려사항
- 안티 타일링 기법: 복잡한 그림에서 효과적
- 2D 환경에서의 작동 여부 불확실
- 두 기법 모두 시도 예정
