# 번호 기입 방법 구상

![screenshot](/docs/image/01_Image_Design_Planning_Documents/10_input_number.ui.png)

## 번호 기입 방법 구상 1번

* 좌측 하단에 `번호 기입하기 ui`를 만들 예정.
* 이 `번호 기입하기 ui`를 누르면 번호 기입이 가능한 모든 기물 위에 `말풍선` ui가 팝업됨.
* 플레이어는 `말풍선` ui를 누르면 번호를 기입할 수 있음.

### 왜 디자인이 말풍선인가?

* ui의 디자인이 말풍선일 필요는 없음.
* 직관적으로 눌러보고 싶게 만들기 위해서 말풍선으로 표기함.
* 느낌표같은 ui로 디자인해도 상관없음.

### 왜 이런 ui를 팝업시키는 방향으로 가는가?

* 새로운 ui를 시각적으로 유저에게 보여주기 위함.
* 번호가 없는 퍼즐칸에만 번호 기입을 할 수 있기 때문에 유저가 이 타일맵을 찾는 수고를 덜기 위함.

## 번호 기입 방법 구상 2번

* `번호 기입하기 ui`를 누르면 번호 기입이 가능한 기물의 타일이 빛남.
* 유저는 빛나는 타일을 누르면 번호 기입을 할 수 있는 ui에 팝업됨.

## 번호 기입 방법 회의 결과

* 1번으로 진행하는 방향을 긍정적으로 검토.
* 일단, 디자이너와의 회의가 있어야 될 것 같음.
* 무조건 말풍선일 필요가 없기 때문에 디자이너의 의견을 반영해서 자연스러운 방향으로 진행하면 될 것 같음.
* 프로토타입 단계에서는 타일을 누르는 방향으로 진행

### 1차 진행 방향

* 기물들을 잔뜩 배치하고 말풍선을 잔뜩 배치해서 어떻게 보이는지 시각적으로 확인하는 과정이 필요.
