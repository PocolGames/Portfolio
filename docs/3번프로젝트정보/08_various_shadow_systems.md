# 여러가지 그림자 시스템

## 그림자 디자인에 대해서

림자는 가짜 그림자와 진짜 그림자로 구분할 수 있다.
가짜 그림자는 구현이 쉽고 간단한 반면, 진짜 그림자는 더 자연스럽지만 추가적으로 구현해야 할 요소가 많다.

각 방식은 장단점이 있으므로, 개발을 진행하면서 게임에 어떤 그림자가 더 적합한지 결정해야 한다.
