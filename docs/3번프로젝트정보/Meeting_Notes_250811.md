# 타일 투시 문제 검토 회의록

## 주요 논의사항 및 결정사항

### 1. 투시 불일치 문제 확인
현재 상황
- 기물들의 투시와 타일맵의 투시가 서로 맞지 않는 문제 발견
- 타일맵 자체의 투시 설정이 부정확한 상태

해결 방향
- 최적화된 기물 투시 방법 연구 필요
- 디자이너 담당 업무로 분류

### 2. 타일맵 투시 적용 가능성 검토
기술적 제약사항
- 타일맵 투시는 시스템상 변경 불가능
- 적용 시 모든 관련 상태를 전면 수정해야 하는 상황

UX 관점 검토 결과
- 투시 적용으로 인한 원거리 타일맵 크기 축소가 사용자 경험에 부정적 영향
- UX적으로 부적절하다고 판단

## 결론
타일맵 투시 변경은 기술적 제약과 UX 문제로 인해 진행하지 않기로 결정. 대신 기물 투시를 타일맵에 맞춰 조정하는 방향으로 해결 예정.
