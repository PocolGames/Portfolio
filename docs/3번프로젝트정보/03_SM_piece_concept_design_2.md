# 길드, 모험가 컨셉 / 유료 기물 디자인 초안

![Screenshot](/docs/image/04_Image_Design_Log_Documents/05_sm_piece_concept_design.png)

## 디자인 피드백

* 직관적으로 어떤 기물인지 알아보기가 힘듬.
* 기본적으로 기물 디자인을 할 때, 배치는 다음과 같이 배치.
* 폰, 나이트, 비숍, 룩, 퀸, 킹
* 위와 같은 순서는 공식적인 체스 기물의 점수와 같음.

## 기타

* 길드나 모험가컨셉으로 진행했을 때, 어떤 기물인지 바로 파악 가능한 디자인이 나올지 모르겠음.
* 킹과 퀸에 대해서 그냥 화려하게 하는 것 말고 떠오르지 않음.

## 1차 디자인 가이드라인

* 폰: 커다란 가방을 매고 있는 짐꾼. 좀 작음.
* 나이트: 말을 타고 있음, 궁수
* 비숍: 마법사. 책이나 지팡이.
* 룩: 망치들고 있음
* 퀸: 갑옷, 엄청 강해보이는 여자, 레이피어
* 킹: 갑옷, 엄청 강해보이는 남자, 대검이랑 방패
