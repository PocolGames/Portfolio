# 회의록

## 회의 내용

각자 Unity Package Manager를 설치하고 진행.

기물 뽑기 시스템을 간소화할 필요성이 있어보임. 기본 체스 공격 방식으로 변경하지만, 기존 공격 방식을 코드에 남겨놔서 추후에 코드 업데이트를 할 수 있도록 진행함. 남겨놓을 코드는 다음과 같음.

``` markdown
# 기물 전투 방식

[기물 A가 기물 B를 공격할 때]

1. B의 체력을 A의 공격력만큼 줄인다.
   → B.체력 = B.체력 - A.공격력

2. A의 체력을 B의 공격력만큼 줄인다.
   → A.체력 = A.체력 - B.공격력

3. A의 체력이 0 이하이면 A는 파괴된다.

4. B의 체력이 0 이하이면 B는 파괴된다.

## 전투 후, 두 기물의 체력이 0이 아닐 때

두 기물 모두 파괴되지 않는다.
이때, 이동 경로에서 도착 지점(상대 기물의 칸)의 이전 칸에 위치한다.
나이트의 경우, 직선 1칸 대각선 1칸으로 이동을 하기 때문에 직선으로 한 칸인 위치로 마무리하게 된다.
```

체스 이미지를 교체하기 쉽게 만들어야됨.
