# 기본 기물 디자인 구체화, 스타일 태그 확정

![Screenshot](/docs/image/04_Image_Design_Log_Documents/04_dj_normal_Piece_concept_design.png)

## 회의 인트로

* 디자인 태그 전달.
* 태그는 다음과 같음.
* `#중세`, `#궁정`, `#기사`, `#왕궁`
* 프로토타입을 위한 임시 이미지이기 때문에 애니매이션 제작 금지.
* 애니메이션은 나중에 집중해서 할 계획.

## 스타일 분석

* 지금 디자인을 작업할 때, 다음과 같은 가이드라인을 따를 것.
* 이 게임의 전체적인 스타일 컨셉은 정해져있지 않다.
* 초반에는 통일된 컨셉을 진행하는 것이 좋다고 판단하였음.
* 그래서, 이 게임의 초기 컨셉은 중세 이세계물로 진행.

## 앞으로의 진행

* 기물들의 디테일을 추가하지 않아도 됨.
* 위와 똑같은 내용으로 어차피 프로토타입이기 때문에 나중에 수정할 시간이 많음.
* 하지만, 백팀과 흑팀의 기물 구분을 해줘야됨.
* 백팀 기물 디자인 세트 1개, 흑팀 기물 디자인 세트 1개 완성 후, 타일맵 디자인으로 넘어갈 것.

## 디자인 패드백.

* 킹의 무기를 수정했고, 디테일한 피드백은 나중에 진행.
