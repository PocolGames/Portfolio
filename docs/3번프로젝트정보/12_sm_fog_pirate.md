# 안개 타일 - 해적

## 추가 설명

문서에서 왼쪽 이미지가 숨겨진 시야 안개 타일, 오른쪽이 반시야 안개 타일이다.
구분이 가지 않을 경우 파일명 확인 바람. _(normal = 숨겨진 시야, half = 반시야)_

### 반시야 타일을 추가적으로 디자인하는 이유

단순히 투명도 조절을 하면 문제가 없는 반시야 타일이지만, 스킨 컨셉에 따라서 반시야 안개 타일의 디자인이 숨겨진 시야 안개 타일의 디자인과 완전히 다를 수가 있기 때문이다.

예를 들면, 안개 타일이 가시 덩굴일 경우, 반시야 타일은 부셔지거나 조각나있는 가시 덩굴 디자인일 수도 있기 때문에 구분을 함.
