# 맵 디자인 레퍼런스 참고 및 세부 디자인 설명

## 설명

유니티의 아이소메트릭 **타일맵 시스템**을 사용하여 맵을 만들 계획.

그리는 방향이나 각도는 다음 내용을 참고.

![ScreenShot](/docs/image/01_Image_Design_Planning_Documents/00_isometric_grid_sample_1.png)

위 이미지는 `256x256` 사이즈를 가지고 있는 이미지. 여백을 유지한 채로 위 파일을 뒤 레이어로 깔고 작업을 진행하면 됨.
중심점이 매우 중요함.

하지만, 이 캔버스 사이즈가 디자인함에 있어서 제한 사항이 있다면 더 큰 캔버스를 사용해도 되나 비율을 유지해야 됨.

위 구도를 지키고 추가적으로 디자인을 진행하면 됨. 기본적으로는 아래로만 넓히는 방향으로 진행되며, 필요 시 옆과 위로도 넓힐 수 있지만 권장하지는 않음.

![ScreenShot](/docs/image/01_Image_Design_Planning_Documents/01_isometric_grid_sample_2.png)

아래로만 넓힌 타일맵.

[그리드 디자인 설명 예시](/docs/Design_Plan/02_design_grid_explain.md)에서 설명했던 것처럼 알아서 가려지는 부분도 있고 여유롭게 디자인을 해도 유니티 내에서 자연스럽게 처리됨.
