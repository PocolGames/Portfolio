# Portfolio

UI/UX Developer에서 PM으로의 성장 과정을 담은 포트폴리오입니다.

## 🌐 Website

[포트폴리오 보기](https://본인아이디.github.io/portfolio/)

## 📋 프로젝트 구성

1. **소울디바이스** - UI/UX 개발 → PM 전환
2. **UI/UX 재정비** - 개발 회고 및 체계화
3. **체스도쿠** - 사업/마케팅 전략 중심 PM
4. **슬롯 꼬치** - AI 활용 초고속 개발

- JavaScript (Vanilla)
